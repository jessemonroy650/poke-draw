Number of Point

Please select the number of points of the item to draw.
  • 1 = dot
  • 2 = line
  • 3 = triangle
  • 4 = box, or similar


 

Size of Point

Please select the width of drawing item (line, stroke, etc).
  • 1 = thin
  • 2 = normal
  • 4 = thick
  • 8 = thicker
